전국 경찰 순찰 안전지도

순찰경로 최적화와 1년 규모 가상순찰 시뮬레이션 결과를 결합하여 전국 도로 구간별 복합 안전지수(0~100)를 산출하고, 치안 사각지대를 사전 진단하는 시각화 웹.

1. 권역별 안전지도

전국 16,827,281개 도로 구간을 6개 권역으로 나누어 제공합니다. 안전지수는 전국 공통 산식으로 일괄 산출되어 권역 간 직접 비교가 가능하며, 권역 구분은 표시·배포 단위입니다.

2. 전국 데이터 현황

안전지수(0~100) = 0.5 × 순찰방문빈도 + 0.3 × 응답접근성(관서→지점 도달시간) + 0.2 × 시설커버리지(CCTV·비상벨)
사각지대 우선도 = 정규화 인구수요 × ( 100 − 안전지수 ) / 100

인구 수요(생활인구·100m 격자)는 높은데 순찰·접근성·시설을 종합한 안전지수가 낮은 구간, 곧 '수요 대비 보호 공급이 얇은' 구간을 사각지대 우선순위로 도출합니다. (뺄셈 잔차가 아닌 exposure × vulnerability 곱)

도로 색상짙은 빨강(위험) → 보라(중간) → 파랑(안전), 6개 이산 구간. 도로(엣지)에만 채색.
상세 로드줌인 후 화면을 3초 이상 정지하면 해당 영역만 로드(과부하 방지). 줌 단계별로 큰길 → 세부길 순으로 표시.

전국 경찰 순찰 안전지도 — 순찰경로 최적화·가상순찰(ABM) 시뮬레이션 기반 도로 안전지수 시각화 안전지수는 전국 공통 산식(0~100)이며, 권역 구분은 표시·배포 단위입니다. 연구·분석용 산출물로 실제 치안 상황과 다를 수 있습니다.
